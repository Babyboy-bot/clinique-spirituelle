1️⃣ INSTALLER LES DÉPENDANCES
   Ouvrir PowerShell dans ce dossier et taper :
   
   npm install

2️⃣ CONFIGURER LES NUMÉROS
   Créer le fichier .env :
   
   copy .env.example .env
   
   Puis éditer .env avec Notepad et remplir :
   - WAVE_NUMBER=+225XXXXXXXXXX
   - ORANGE_MONEY_NUMBER=+225XXXXXXXXXX  
   - WHATSAPP_SUPPORT_NUMBER=+225XXXXXXXXXX

3️⃣ DÉMARRER LE SERVEUR
   
   npm start
   
   Puis ouvrir dans le navigateur :
   http://localhost:3001
